
# ES6 Basics - Holberton School

## Description
This project covers the fundamental concepts of **ES6 (ECMAScript 2015)**. It focuses on modern JavaScript syntax, improving code readability, and using professional development tools like Babel, Jest, and ESLint.

## Learning Objectives
By the end of this project, I have practiced:
*   Using `const` and `let` instead of `var`.
*   Writing **Arrow Functions**.
*   Working with **Rest and Spread** parameters.
*   Implementing **String Templating**.
*   Creating **ES6 Iterators** and using `for...of` loops.
*   Handling **Computed Property Names** in objects.

## Requirements
*   **Operating System:** Ubuntu 20.04 LTS
*   **Node.js Version:** 20.x.x
*   **Package Manager:** npm 9.x.x
*   **Compiler:** Babel
*   **Linter:** ESLint (Airbnb style)
*   **Testing:** Jest

## Setup
To install the project dependencies, run:
```bash
npm install
```
How to Run:
To execute a specific script (e.g., 0-main.js):
Bash
```
npm run dev 0-main.js
```
To run the tests:
Bash
```
npm test
```
To check for linting errors:

Bash
```
npm run lint
```
