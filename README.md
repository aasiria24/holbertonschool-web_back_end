# Holberton School - Web Back End

## Projects

### Python Variable Annotations
Type annotations in Python 3, duck typing, and mypy validation.
* Directory: `python_variable_annotations`

### Python Async Function
Asynchronous programming with async/await syntax, asyncio, and coroutines.
* Directory: `python_async_function`

### Python Async Comprehension
Asynchronous generators, async comprehensions, and parallel coroutines.
* Directory: `python_async_comprehension`

### Pagination
Pagination techniques using simple page/page_size parameters, hypermedia
metadata, and deletion-resilient pagination on a dataset of popular baby names.
* Directory: `pagination`

### NoSQL
NoSQL databases using MongoDB and PyMongo, including CRUD operations,
aggregation pipelines, and Nginx log statistics.
* Directory: `NoSQL`
