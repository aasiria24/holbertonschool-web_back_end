# ES6 Classes - Holberton School

## Description
This project focuses on **Object-Oriented Programming (OOP)** in JavaScript using ES6 syntax. It covers how to define classes, handle inheritance, and use advanced concepts like static methods and metaprogramming symbols.

## Learning Objectives
By the end of this project, I have learned:
*   How to define a **Class** and its **Constructor**.
*   How to add **Methods** (Instance and Static) to a class.
*   How to implement **Inheritance** using `extends` and `super`.
*   How to use **Getters and Setters** for data validation.
*   Understanding **Metaprogramming** with Symbols (`Symbol.species`, `Symbol.toPrimitive`, `Symbol.toStringTag`).

## Requirements
*   **Operating System:** Ubuntu 20.04 LTS
*   **Node.js Version:** 20.x.x
*   **Package Manager:** npm 9.x.x
